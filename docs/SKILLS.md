# Skills

Skills are used during adventures.
Skills can be used for combat and non-combat actions.
Skills are passive.

## Vuex

```
skills : {
  *skillId* : *skillExperience
}

Skill experience is stored as a numerical value in the Vuex data store. Computing levels of a given skill is done as a Vuex getter, or Utility getter.


## Climbing

Climbing is used on ladders, cliffs, buildings, trees.


## Healing

Basic first aid and medicine for post-combat wounds.

## Tracking

Following animals or persons, typically through woodlands, not for an urban environment.
